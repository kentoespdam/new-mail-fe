# Issue #5.1: Debounced Auto-save Engine

**Epic:** EPIC 5 - Smart Composer & Auto-save Logic  
**Priority:** High  
**Status:** Pending

---

## Description

Mengimplementasikan fitur simpan otomatis (Auto-save) agar Staff tidak kehilangan draf surat saat terjadi gangguan koneksi.

## Technical Details

- **Mutation Method**: `POST /api/v1/mails` (new draft), `PUT /api/v1/mails/{id}` (update draft)
- **Debounce Library**: `lodash.debounce` or custom hook
- **State Management**: Track draft ID to determine POST vs PUT operation
- **Status Indicators**: "Menyimpan...", "Tersimpan otomatis", "Gagal menyimpan"

## Tasks

- [ ] Set up `useMutation` from TanStack Query for draft creation (`POST`) and update (`PUT`)
- [ ] Implement debounce logic (2-3 seconds) using `lodash.debounce` or custom hook
- [ ] Create status indicator component in form corner showing save state:
  - "Menyimpan..." (Saving...)
  - "Tersimpan otomatis" (Auto-saved)
  - "Gagal menyimpan" (Failed to save)
- [ ] Ensure draft ID from new creation is stored in state to enable subsequent `PUT` operations
- [ ] Handle error states and retry logic for failed saves

## Acceptance Criteria

- [ ] Draft auto-saves to server every 3 seconds after user stops typing
- [ ] User does not need to press "Simpan" button manually
- [ ] Status indicator accurately reflects save state
- [ ] Draft ID persists correctly to switch from POST to PUT after initial creation
- [ ] Failed saves display appropriate error messages

## Implementation Notes

### TanStack Query Pattern

```typescript
const createDraftMutation = useMutation({
  mutationFn: (data: MailDraft) => mailService.createDraft(data),
});

const updateDraftMutation = useMutation({
  mutationFn: ({ id, data }: { id: string; data: MailDraft }) => 
    mailService.updateDraft(id, data),
});
```

### Debounce Implementation

```typescript
const debouncedSave = useCallback(
  debounce((draftData: MailDraft) => {
    if (draftId) {
      updateDraftMutation.mutate({ id: draftId, data: draftData });
    } else {
      createDraftMutation.mutate(draftData);
    }
  }, 3000),
  [draftId, createDraftMutation, updateDraftMutation]
);
```

### Status Indicator States

```typescript
type SaveStatus = 'idle' | 'saving' | 'saved' | 'failed';
```
