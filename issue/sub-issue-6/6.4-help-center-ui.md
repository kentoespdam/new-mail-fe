# Issue #6.4: Documentation & Help Center UI

**Epic:** EPIC 6 - User Onboarding & Help  
**Priority:** Medium  
**Status:** Pending

## Description

Menyediakan area khusus untuk membaca panduan penggunaan secara mendalam tanpa harus menjalankan *tour*.

## Technical Details

- **Component**: `HelpCenterModal` using ShadcnUI Dialog
- **Content**: Categorized help topics (Admin, Writing Mail, Managing Folders)
- **Format**: Markdown or structured content with visual aids (icons/images)
- **Responsive**: Comfortable reading experience on all screen sizes

## Tasks

- [ ] Create `HelpCenterModal` component using ShadcnUI Dialog
- [ ] Organize help content by categories:
  - **"Tentang Admin"**: How to manage mail types, categories, file rules
  - **"Menulis Surat"**: Composing, sending, replying, forwarding
  - **"Mengelola Folder"**: Creating, organizing, navigating folders
- [ ] Use visual elements (icons/images) for easier comprehension
- [ ] Ensure modal is responsive and comfortable to read
- [ ] Add search/filter functionality (optional enhancement)

## Acceptance Criteria

- [ ] Users can find answers to their confusion independently
- [ ] Modal is responsive and comfortable to read on all screen sizes
- [ ] Help content is well-organized by category
- [ ] Visual aids enhance understanding

### Reusable Help Section Component

```typescript
interface HelpSectionProps {
  title: string;
  icon: React.ElementType;
  steps: string[];
}

function HelpSection({ title, icon: Icon, steps }: HelpSectionProps) {
  return (
    <div className="rounded-lg border p-4">
      <div className="flex items-center gap-2 mb-3">
        <Icon className="h-5 w-5 text-amber-500" />
        <h3 className="font-semibold">{title}</h3>
      </div>
      <ol className="list-decimal list-inside space-y-2 text-sm text-slate-700">
        {steps.map((step, index) => (
          <li key={index}>{step}</li>
        ))}
      </ol>
    </div>
  );
}
```
