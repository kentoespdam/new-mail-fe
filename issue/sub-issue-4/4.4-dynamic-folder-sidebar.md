### **Issue #4.4: Dynamic Folder Sidebar (Tree View)**

**Epic**: EPIC 4 - The Mailbox Engine (Classic vs Modern)
**Parent Issue:** [Issue #4](../0-blueprint/4-mailbox-engine.md)

**Description:**
Menampilkan struktur folder surat secara dinamis dari API, mendukung sub-folder untuk tampilan klasik.

**Tasks:**
- [ ] Konsumsi endpoint `GET /api/v1/mail/folders`.
- [ ] Bangun komponen **Tree Navigation** (khusus Classic) dan **Flat List** (khusus Modern).
- [ ] Tampilkan *badge* jumlah pesan belum dibaca (Unread Count) di samping nama folder.
- [ ] Gunakan warna **30% Indigo-900** untuk latar belakang sidebar agar kontras dengan konten.

**Acceptance Criteria:**
- Sidebar mencerminkan struktur folder yang ada di backend.
- Navigasi folder memperbarui konten list surat secara otomatis.
