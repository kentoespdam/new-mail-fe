### **Issue #4.1: Layout Switcher & Preference Sync**

**Epic**: EPIC 4 - The Mailbox Engine (Classic vs Modern)
**Parent Issue:** [Issue #4](../0-blueprint/4-mailbox-engine.md)

**Description:**
Membangun sistem untuk beralih antara tampilan **Klasik** dan **Modern** serta menyimpannya secara permanen di profil pengguna Appwrite.

**Tasks:**
- [ ] Tambahkan field `theme_layout` (enum: `classic` | `modern`) pada `account.updatePrefs()` Appwrite.
- [ ] Buat **Context Provider** atau **Zustand Store** untuk mengelola state layout secara global.
- [ ] Implementasikan *toggle switch* di Header atau Sidebar untuk berpindah layout secara *real-time*.
- [ ] Pastikan saat aplikasi di-refresh, layout tetap sesuai dengan pilihan terakhir pengguna.

**Acceptance Criteria:**
- Pengguna bisa berpindah tampilan tanpa kehilangan sesi.
- Pilihan layout tersimpan di database Appwrite.
