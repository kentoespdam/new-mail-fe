### **Issue #1.2: Tailwind Design System (Rule 60:30:10)**

**Epic**: EPIC 1 - Init & Infrastructure
**Parent Issue:** [Issue #2](../0-blueprint/1-init-infrastructure.md)

**Description:**
Konfigurasi sistem desain global menggunakan Tailwind CSS dengan menerapkan aturan proporsi warna 60:30:10 untuk identitas visual PERUMDAMTS.

**Tasks:**
- [ ] Update `tailwind.config.ts` untuk mendaftarkan variabel warna:
  - **Base (60%)**: `slate-50` (untuk background utama).
  - **Secondary (30%)**: `indigo-900` (untuk sidebar dan navigasi).
  - **Accent (10%)**: `amber-500` (untuk aksi utama/CTA).
- [ ] Install ShadcnUI dan inisialisasi komponen dasar (Button, Input, Card).
- [ ] Buat file `src/app/globals.css` untuk menetapkan variabel CSS `:root` agar konsisten dengan palet warna instansi.

**Acceptance Criteria:**
- Variabel warna dapat digunakan via class Tailwind (e.g., `bg-primary`, `text-accent`).
- Komponen ShadcnUI terintegrasi dengan tema warna.
